wxMimeType

This is currently just a prove of concept, so don't expect too much.

wxMimeType allows to retrieve and to set all the mime type information of file:

  • Gets the mime type of a file through filename and/or content
  • Reads all and some the data of a mime type, e.g. application, icon
  • Writes the data of a mime type, creates it if necessary

wxMimeType uses the following data structure: 

struct wxMimeTypeData {
    wxString filename; // not used so far
    wxString extension;
    wxString magicId; // not used so far
    wxString creator; // not used so far
    wxString description;
    wxString appname;
    wxIcon icon; // not used so far
};

wxMimeType::Add

bool Add (const wxString& mimetype, wxMimeTypeData* data, bool standard = false)

Adds an association of a file type to a mime type. Returns true if successful.

Currently works only under Windows and only with file extension.

wxMimeType::Remove

bool Remove (wxMimeTypeData* data)

Removes an association of a file type to a mime type. Returns true if successful.

Currently works only under Windows and only with file extension.

wxMimeType::Exists

bool Exists (wxMimeTypeData* data)

Returns true if the mime type of this association exists.

Currently works only under Windows and only with file extension.

wxMimeType::Match

bool Match (wxMimeTypeData* data)

Returns true if the mime type of this association exists and the application name matches.

Currently works only under Windows and only with file extension.

wxMimeType::IsStandard

bool IsStandard (wxMimeTypeData* data)

Returns true if the mime type of this association exists, the application name matches and the application is the standard for this mime type.

Currently not implemented.
