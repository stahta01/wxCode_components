mbrola.ini file format

wxMbrola tries to find an mbrola.ini file in the current directory or Windows directory. This file allows the application to specify useful values, which don't affect wxMbrola directly but can be used in an application to tailor the voice parameters, etc. This facility might be considered too application-specific to be in wxMbrola and could be removed in the future.

[GENERAL]
Path=<string> ; Specifies a path (what for?)

[PITCH]
Min=<float>   ; E.g. 0.2
Max=<float>   ; E.g. 8.0

[TIME]
Min=<float>   ; E.g. 0.2
Max=<float>   ; E.g. 8.0

[VOICE] ; Frequency. What about having default freq. (16000)?
Min=<float>   ; E.g. 8000
Max=<float>   ; E.g. 32000

[DATABASE]
Default=<string>
