Compiling the wxTTS libraries and dictionary

To compile all the library files, and the dictionary used by wxFreePhone, you can type this in a DOS box:


cd src
nmake -f makefile.vc all

However, you may wish to only compile some of the libraries, in which case you can use one or more of these targets:

  • just_win32: Make WIN32 libraries, debug and release
  • just_wxwin: Make wxWin libraries, debug and release
  • just_mfc: Make MFC libraries, debug and release
  • dictionary: Make the dictionary

You can also compile the wxMbrola and wxFreePhone libraries individually using the makefile.vc files provided in each directory under src.

Note that all makefiles have a 'clean' target to remove target and intermediate files.
