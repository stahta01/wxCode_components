wxTTS Samples

WIN32 samples

samples/win/simple shows a dialog that allows you to enter text and speak it or write it to a phoneme file (test.pho) or a wave file (test.wav).

samples/win/freephone only uses wxFreePhone, and allows you to translate text to a phoneme file (test.pho).

MFC samples

samples/win/simple shows a dialog that allows you to enter text and speak it or write it to a phoneme file (test.pho) or a wave file (test.wav).

samples/win/mbrola is an adaptation of code supplied by the authors of Mbrola, and allows setting of various Mbrola parameters.

wxWindows samples

samples/win/simple shows a dialog that allows you to enter text and play it to the sound card or a wave file. You can also play a phoneme file.
