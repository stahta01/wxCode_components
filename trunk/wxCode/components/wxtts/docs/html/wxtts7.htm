Compiling the wxTTS samples

To compile all the samples in release mode, you can type this in a DOS box:


cd samples
nmake -f makefile.vc all

However, you may wish to only compile some of the samples, in which case you can use one or more of these targets:

  • samples_win32: Make WIN32 executables, release only
  • samples_wxwin: Make wxWindows executables, release only
  • samples_mfc: Make MFC executables, release only

You can also compile the samples individually using the makefile.vc files provided in each directory under samples. You can alternatively use the provided project files if you wish.

Note that all makefiles have a 'clean' target to remove target and intermediate files.

See wxTTS Samples for a brief description of the samples.
