wxSplitWindow

wxSplitWindow allows to split a window into 2 subwindows either horizontal or vertical with a sash in between. It's similar to wxSpitterWindow but it doesn't destroy any window if unsplit. So the 2 partial windows can always be access regardless if split or unsplit.

Parameters

parent

      Parent window. Must not be NULL.

      Window identifier. A value of -1 indicates a default value.

      Window position.

size

      Window size. If the default size (-1, -1) is specified then the window is sized appropriately.
