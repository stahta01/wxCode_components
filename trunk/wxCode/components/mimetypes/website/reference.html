wxMimeTypes

wxMimeTypes allows to retrieve and to set all the mime type information of file:

  • Gets the mime type of a file through filename and/or content
  • Reads all and some the data of a mime type, e.g. application, icon
  • Writes the data of a mime type, creates it if necessary

wxMimeTypes::GetMimeType

wxString GetMimeType(const wxString& filename) const

Returns the first found mime type of a file. First the filename (complete or extension) is tried. If no mime type is found the content of the file is analysed. If none is found an empty string is returned.

wxMimeTypes::GetMimeTypeByContent

wxString GetMimeTypeByContent(const wxString& filename) const

Returns the first found mime type of a file. Only the content is analysed. If none is found an empty string is returned.

wxMimeTypes::GetMimeTypeByFilename

wxString GetMimeTypeByFilename(const wxString& filename) const

Returns the first found mime type of a file. Only the filename (complete or extension), is tried. If none is found an empty string is returned.

wxMimeTypes::GetApplication

wxString GetApplication(const wxString& mimetype) const

Returns the default application name (including full path) which handles this mime type.

wxMimeTypes::SetApplication

bool SetApplication(const wxString& mimetype, const wxString& filename)

Sets the default application name (any kind of path) of this mime type. If the mime type couldn't be updated (e.g. it doesn't exists) or if the application's full path couldn't be determined, false is returned.

This function should only be used by applications to set themselfs as the standard application of a mime type, otherwise SetData should be used.

wxMimeTypes::GetData

wxMimeTypeData* GetData(const wxString& mimetype) const

Returns the full data of this mime type.

wxMimeTypes::SetData

bool SetData(const wxString& mimetype, wxMimeTypeData* data, bool force = false)

Sets the full data of this mime type. If the mime type doesn't exists and force is true, the mime type is created. If it couldn't be updated or created, false is returned.

wxMimeTypes::GetIcon

wxIcon GetIcon(const wxString& mimetype) const

Returns the icon which is associated with this mime type.

wxMimeTypes::SetIcon

bool SetIcon(const wxString& mimetype, const wxIcon& icon)

Sets the icon of this mime type. If the mime type couldn't be updated (e.g. it doesn't exists), false is returned.
