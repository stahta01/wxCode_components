Keybinder

Synopsis

A generic system to bind one or more keyshortcut to the commands you want (menu commands or whatever you like). A pre-built panel and a complete system to view/remove/add/edit command shortcuts is included. Also key profiles are supported and a complete load/save system based on wxConfig is present. Integrates with your applications with few modifications and allows the users to customize all the commands he wants, bypassing the limitation of a single keybind to each command.
