wxString wxLocalize(const wxString& searchPath, const wxString& fullName, const wxString& lanugage)

Searches the file specified by its full filename in the search paths for the language. Returns the name of the found file, else returns empty string.

searchPath

      List of paths, separated by wxPATH_SEP (';'), to be searched

fullname

      Full filename (name and extension) to be searched.

lanugage

      Language (canonical name) to be used
