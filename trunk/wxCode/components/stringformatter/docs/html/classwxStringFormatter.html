Detailed Description

This is the main class to use for text formatting.

Order of evaluation is:
left to right
Block > Function parameter > Function > Symbol
The string:
" FUNC($sym3) blah $sym4 [blah [$sym1][$sym2]]\n " would be evaluated:
sym1 > sym2 > sym3 > FUNC > sym4
