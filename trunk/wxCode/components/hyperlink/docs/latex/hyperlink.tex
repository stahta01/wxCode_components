\section{\class{wxHyperLink}}\label{wxhyperlink}

This class contains a static text element which links to an URL. Clicking on
the URL activates the standard browser (via mimetype) and opens the URL. No 
checks on the URL is done.

\wxheading{Derived from}

\helpref{wxStaticText}{wxstatictext}
