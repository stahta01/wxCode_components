Sample demo's description:
-----------------------------

./samples/Python/pythonEN.demo - this sample demo demostrates creation of simple skin, lexer
    and few Python samples.

./samples/Python/pythonSK.demo - this sample demo demostrates creation of simple skin, lexer,
    few Python samples and translation into Slovak language.

To launch these demos, simply delete/rename the autostart.conf file and select demo file
in wxDemoViewer application.
