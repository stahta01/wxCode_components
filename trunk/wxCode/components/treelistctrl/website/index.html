wxTreeListCtrl

Synopsis

A multi column tree control.

Description

wxTreeListCtrl is able to show a tree with multiple colums. The main column is shown as a tree, the others just as lists.

  • Sorting after any column, incl. native numeric ordering
  • Keyboard navigation, positioning via characters (main column)
  • Show/hide any column (except main column)

The API reference can be found here.
