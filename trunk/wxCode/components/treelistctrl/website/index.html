wxCode :: wxTreeListCtrl

Description & documentation

[wxCode::]wxTreeListCtrl A multi column tree control. It is able to show a tree with multiple colums. The main column is shown as a tree, the others just as lists.

  • Sorting after any column, incl. native numeric ordering
  • Keyboard navigation, positioning via characters (main column)
  • Show/hide any column (except main column)
  • various attributes configurable for each row and cell: color, font, picture...

Here the API reference documentation.

NOTE: This is NOT the same control as introduced in wxWidgets 2.9

Although the original intention was probably to make wxCode::wxTreeListCtrl a standard wxWidgets control, the wxWidgets::wxTreeListCtrl introduced with wxWidgets 2.9 is a new code base not related to wxCode.
